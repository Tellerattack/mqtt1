# mqtt1
基于mqtt的遥控器，在app上点击按钮，将码（空调码，电视码，风扇码）发送到mqqt，mqtt通过WiFi发给esp8266,esp8266解析转为红外，发出去。实现手机控制电视，风扇，空调。现已匹配到索尼电视，格力空调，奥美克风扇。
